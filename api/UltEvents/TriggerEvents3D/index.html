TriggerEvents3D Class

Summary

Holds UltEvents.UltEvents which are called by various UnityEngine.MonoBehaviour trigger events: UltEvents.TriggerEvents3D.OnTriggerEnter(UnityEngine.Collider), UltEvents.TriggerEvents3D.OnTriggerStay(UnityEngine.Collider), and UltEvents.TriggerEvents3D.OnTriggerExit(UnityEngine.Collider).
